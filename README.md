This project was an extension of my Final Project from 15-112 (https://academy.cs.cmu.edu/sharing/indigoShark086198/embed). I had designed the rules for Achilles Heel Chess last year and written it in Brython. For this project I uploaded the Brython code and had Claude/ChatGPT help me learn how to write it using JS so that it could be played on websites locally. Then I used Claude/ChatGPT to help me set up a backend and enable online play by leveraging websockets. To use the project simply go to this vercel link: https://achilles-heel-online.vercel.app/ and you will be able to play it. For the rules either look at this doc: https://docs.google.com/document/d/1t5QznlGl9KfNB7o9SDkvk10Hlpdo89gkAm8lncmND00/edit?tab=t.0 or just read the rules from the website. I am most proud of the online web play and the code for the pawn promotion as those were the more complex aspects of the code. 

You could run it locally as the code does connect to the backend you just need to start the server and just go to the right URL based on your port.
