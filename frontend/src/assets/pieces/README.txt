# Path: AchillesHeelOnline/frontend/src/assets/pieces/README.txt
Place chess piece sprites in this folder. Naming convention:

- white_pawn.png
- white_rook.png
- white_knight.png
- white_bishop.png
- white_queen.png
- black_pawn.png
- black_rook.png
- black_knight.png
- black_bishop.png
- black_queen.png

No king in Achilles Heel variant.

You can use open source chess piece sprites, e.g. Wikimedia Commons or Lichess SVGs. For demo, you may use placeholder icons or emoji.
